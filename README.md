# Праздники

## О проекте
Этот код выдает все праздники и их даты.

## Установка
Скачайте необходмые файлы, затем  используйте pip(или pip3, если конфликты с Python2) для установки зависимостей и установить зависимости 

Установить зависимости командой:
```pip install -r requirements.txt```

## Как запустить
Для запуска у вас уже должен быть установлен Python3

Запустите код командой:
```python main.py```

## Переменные окружения 

Вы должнеы получить api ключ на этом [сайте](https://calendarific.com). Далее полученый ключь засуньте в файл .env
```TOKEN="ваш ключ"```
